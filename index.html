Biography

I am currently a third-year Ph.D. candidate at Fudan University, working under the supervision of Prof. Yu Cheng. My research focuses on Visual Agents and Multi-step Reasoning, with the goal of improving how large models perceive, plan, and reason through structured interaction with external tools.

More broadly, I am interested in reliable large vision-language models. My recent work studies hallucination mitigation, data imbalance, and evaluation frameworks for long and step-level reasoning. I have worked with Shanghai Artificial Intelligence Laboratory and The Chinese University of Hong Kong on research spanning visual agent trajectories, reasoning evaluation, and LVLM reliability.

Selected Publications

AdaReasoner: Dynamic Tool Orchestration for Iterative Visual Reasoning

ICLR 2026 | Mingyang Song*, Haoyu Sun*, Jiawei Gu*, Linjie Li*, Ranjay Krishna, Yu Cheng

Proposed a unified framework for data curation, reinforcement learning, and evaluation of tool-planning models. The model exhibits self-adaptive tool use and achieves substantial gains on visual spatial planning tasks.

Paper / Project / Code

PRMBench: A Fine-grained and Challenging Benchmark for Process-Level Reward Models

ACL 2025 Main | Mingyang Song, Zhaochen Su, Xiaoye Qu, Jiawei Zhou, Yu Cheng

Introduced a comprehensive benchmark for process-level reward models with 6,216 curated samples and 83,456 step-level labels, revealing key limitations in current PRMs.

Paper / Project / Code

From Head to Tail: Towards Balanced Representation in Large Vision-Language Models through Adaptive Data Calibration

CVPR 2025 | Mingyang Song, Xiaoye Qu, Jiawei Zhou, Yu Cheng

Conducted an in-depth analysis of the long-tail problem in current LVLM training data and proposed a fine-grained adaptive calibration framework for more balanced representation learning.

Paper / Project / Code

Mitigating Multilingual Hallucination in Large Vision-Language Models

TOMM | Xiaoye Qu*, Mingyang Song*, Wei Wei, Jianfeng Dong, Yu Cheng

Developed an early approach for mitigating multilingual hallucinations in LVLMs and analyzed the main factors that drive non-English hallucination behavior.

Paper / Code
